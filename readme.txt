== Description ==

Create HTML text effects by surrounding your line by shortcodes. Nice selection of cool effects available!

See [WordPress Text FX Shortcodes homepage](https://github.com/tzri/wptexteffects) for more info.

Example:
[blink]this text blinks[/blink]
